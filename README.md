# 🚌 SafeGo School

> Sistema de gestión y seguimiento en tiempo real para el transporte escolar.

## 📝 Descripción del Proyecto
SafeGo School es una solución tecnológica desarrollada como proyecto formativo para el programa **Tecnólogo en Análisis y Desarrollo de Software (ADSO) - SENA**. 

El objetivo es brindar tranquilidad a los padres de familia y optimizar la gestión logística de las instituciones educativas y conductores de rutas escolares.

---

## 🚀 Funcionalidades Principales

* 👨‍👩‍👧 **Módulo Padres:** 
  * Notificaciones de abordaje y desembarque del estudiante.
  * Monitoreo de la ruta escolar en tiempo real.
  * Estimación de tiempo de llegada.

* 🏫 **Módulo Institución Educativa:**
  * Control y gestión de rutas escolares, conductores y vehículos.
  * Asignación de estudiantes a rutas activas.

* 🚍 **Módulo Conductores:**
  * Control del número de estudiantes a bordo.
  * Gestión de rutas programadas y asistencia.

---

## 🛠️ Tecnologías y Herramientas
* **Control de Versiones:** Git & GitHub
* **Metodología:** Análisis y Desarrollo de Software (ADSO)
